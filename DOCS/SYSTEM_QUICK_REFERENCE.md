## 🎁 COMBO BONUS SYSTEM

### How It Works
When **2+ sources agree** on same signal (BUY or SELL):

```
4/4 sources align   → UNANIMOUS (1.30-1.40x boost)
3/4 sources align   → SUPER_STRONG (1.20-1.25x boost)
2/4 sources align   → STRONG (1.10-1.15x boost)
1/4 sources align   → WEAK (1.00x, no boost)
```

### The 4 Sources
1. **Scanner** (Momentum Scanner) - 0.25 weight
2. **ML** (LSTM Inference) - 0.25 weight
3. **RL** (Policy Learning) - 0.25 weight
4. **RPG** (Agent Arena Vote) - 0.25 weight

### Example
```
Signal generated by RPG agents: BUY @ 0.68 confidence
Scanner also says: BUY @ 0.80
ML also says: BUY @ 0.75
RL also says: BUY @ 0.78

Result: 4/4 sources = UNANIMOUS
Combo bonus: 0.68 * 1.35 = 0.92 confidence (capped 0.95)
```

## 🧪 QUALITY GATING: 5 LAYERS

```
Signal enters gating
    ↓
Layer 1: Base confidence > 0.30?
    ↓
Layer 2: 4-source consensus active? (apply combo bonus)
    ↓
Layer 3: Regime threshold met? (e.g., 0.40 in CHOPPY)
    ↓
Layer 4: Source agreement? (2+ sources agree on direction?)
    ↓
Layer 5: Final confidence valid? (0.0 < confidence < 1.0?)
    ↓
✅ PASS → Ready for execution
❌ FAIL → HOLD (no trade)
```

```

Level 10: multi_timeframe_fusion
Level 15: pattern_memory
Level 20: ability_combo_chains
Level 30+: legendary_abilities
```

### Policy Learning
After each trade closes:
```
PROFITABLE TRADE   → Agent confidence +0.05
LOSING TRADE       → Agent confidence -0.10
BOUNDS             → [0.1, 0.95] (can't go below/above)
```

Over time, winning agents get stronger signals, losing patterns suppressed.

## 🔍 SIGNAL GENERATION: STEP-BY-STEP

### Input: Market Data
```json
{
  "symbol": "BTC/USD",
  "high": 45200,
  "low": 44800,
  "close": 45100,
  "volume": 2500,
  "adx": 32,
  "rsi": 62,
  "momentum": 0.55
}
```

### Step 1: Regime Detection
```
ADX > 25 + momentum > 0 + ATR% > threshold
→ LAMINAR_TREND detected
```

### Step 2: 13 Agents Generate Signals
```
VFMD Agent:          BUY @ 0.82
Convexity Agent:     HOLD @ 0.40
Flow Physics:        BUY @ 0.70
Trend Rider:         BUY @ 0.75
Reversal Master:     HOLD @ 0.50
Support Sniper:      BUY @ 0.68
Breakout Hunter:     BUY @ 0.78
Volume Agent:        BUY @ 0.72
ML Oracle:           BUY @ 0.80
Market Oracle:       BUY @ 0.75
Feature Engineer:    (support role)
Python Strategy:     BUY @ 0.65
Exit Agents:         (neutral)
```

### Step 3: Arena Voting
```
BUY votes: 9/13 agents
Confidence average: 0.74
→ Consensus: BUY @ 0.74
→ Strategy: TREND_FOLLOWING (from Policy Q-values)
```

### Step 4: 4-Source Consensus
```
Scanner confidence:   0.76
ML confidence:        0.80
RL confidence:        0.78
RPG confidence:       0.74
Average:              0.77 (77%)

Source agreement:     4/4 align on BUY
Combo type:          UNANIMOUS
Combo bonus:         1.35x
```

### Step 5: Quality Gating
```
Layer 1: 0.74 > 0.30?        ✅ PASS
Layer 2: Combo bonus active? ✅ 1.35x applied → 0.99 (capped 0.95)
Layer 3: Regime threshold?   ✅ 0.95 > 0.35 (LAMINAR_TREND min)
Layer 4: Source agreement?   ✅ 4/4 sources agree
Layer 5: Final check?        ✅ 0.95 ∈ [0.0, 1.0]

Result: ✅ GATED SIGNAL PASS
```

### Step 6: Position Sizing
```
Account risk: $1000 (1% of $100k account)
Entry: $45,100
Stop:  $44,800 ($300 risk)
Size = ($1000 / $300) × 0.95 confidence
     = 3.33 × 0.95
     = 3.16 contracts
```

### Step 7: Trade Execution
```
BUY 3.16 BTC @ $45,100
Stop Loss: $44,800
Target 1: $45,500 (small profit)
Target 2: $45,900 (medium profit)
Target 3: $46,300 (large profit)
```

### Step 8: Trade Closes
```
Result: +$950 profit

Agents that correctly predicted BUY:
  +0.05 confidence each

Agents that predicted HOLD/SELL:
  -0.10 confidence each
```

### Step 9: Next Cycle
Agents with updated confidence generate slightly better signals.

## 💡 PRO TIPS

1. **Combo Bonus Signals Are Strongest**
   - UNANIMOUS (4/4 sources) align = highest confidence
   - Filter trades to only STRONG+ combos if conservative

2. **Regime Matters More Than Indicators**
   - VFMD works best in LAMINAR_TREND
   - Reversal Master works best in CONSOLIDATION
   - Use regime to weight agent signals

3. **Position Size Discipline**
   - 1% account risk = slow but consistent
   - 5% account risk = faster growth but requires monitoring
   - Start conservative, scale after 100+ profitable trades

4. **Agent Disagreement Is A Signal**
   - If agents vote 50/50, quality gate should HOLD (no trade)
   - If 9+ agree, signal is strong (likely profitable)

5. **Watch Combo Bonus Frequency**
   - 20-30% of signals have STRONG+ combo = healthy
   - >40% = signal quality may be declining (regime change?)
   - <10% = sources may not be aligned well

6. **Q-Value Learning Takes Time**
   - Agent confidence updates slowly (+0.05 per win)
   - After 100 trades, you'll see agents differentiate
   - Don't expect perfect learning in first 50 trades
