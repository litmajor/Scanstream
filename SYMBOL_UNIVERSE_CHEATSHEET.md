# Symbol Universe - Quick Reference Card

## 🎯 What is Symbol Universe?

A centralized dashboard for discovering, analyzing, and managing your trading universe. All in one place.

---

## 🗺️ How to Get There

```
Sidebar → Tools → Symbol Universe
or
Direct URL: /symbol-universe
```

---

## 🔍 5 Main Tabs

### 1. 🔎 Symbols (Discovery)
**What**: Search and browse all trading symbols  
**How**: Type in search bar, adjust filters  
**View**: Table or Grid  
**Action**: Click ⭐ to add to watchlist, 📊 for details, 📈 to trade

### 2. 🔥 Heatmap (Liquidity)
**What**: Best exchange for each symbol  
🟢 = Great | 🟡 = Good | 🔴 = Fair | ⚪ = N/A  
**Why**: Find venues with lowest slippage

### 3. 🔗 Correlation (Portfolio)
**What**: How symbols move together  
🟢 = Move together | 🔴 = Move opposite  
**Why**: Diversify portfolio, hedge positions

### 4. 📊 Volume (Analysis)
**What**: Volume distribution at price levels  
3 views: Profile, History, Anomalies  
**Why**: Spot accumulation zones, unusual activity

### 5. 📌 Watchlists (Organization)
**What**: Custom symbol collections  
**Actions**: Create, edit, delete, share  
**Why**: Organize by strategy or market segment

## ⚡ Quick Actions

| Action | Steps | Result |
|--------|-------|--------|
| Find Symbol | Search bar | See all matches |
| Filter Down | Select filters | See subset matching all criteria |
| Add to Watchlist | Click ⭐ | Symbol added to selected list |
| Create Watchlist | Click ➕ → Enter name | New empty list ready |
| View Details | Click symbol row | Side panel opens with full info |
| Go to Trade | Click 📈 icon | Opens trading terminal |

---

## 🎛️ 6 Filter Types

1. **Exchange** - Which exchange (Binance, Coinbase, etc.)
2. **Asset Class** - Spot, Futures, Perpetual
3. **Market Cap** - Size (Mega, Large, Mid, Small)
4. **Min Volume** - Dollar amount in 24h
5. **Min Liquidity** - 0-100 scale (higher = better)
6. **Volatility** - Price change range

---

## 📊 What The Colors Mean

### Liquidity Heatmap
- 🟢 **80-100**: Excellent (tight spreads)
- 🟡 **60-79**: Good (acceptable spreads)
- 🔴 **40-59**: Fair (wider spreads)
- ⚪ **N/A**: Not available

### Correlation Matrix
- 🟢 **>0.7**: Strong Positive (move together)
- 🟡 **0.3-0.7**: Moderate Positive
- ⚪ **-0.3-0.3**: Weak (independent)
- 🔴 **<-0.7**: Strong Negative (opposite moves)

## 💡 Common Use Cases

### "I want to trade Bitcoin"
1. Search "Bitcoin" or "BTC"
2. Click the symbol row
3. View liquidity heatmap to pick exchange
4. Click 📈 → Trade

### "I need a diversified portfolio"
1. Create watchlist "Diversified"
2. Go to Correlation tab
3. Add symbols that don't correlate
4. Add to watchlist

### "Find good execution venues"
1. Go to Heatmap tab
2. Look for all 🟢 (excellent) or 🟡 (good)
3. Those exchanges = best for trading

### "Spot unusual activity"
1. Go to Volume tab
2. Click Anomalies view
3. See recent volume spikes/drops
4. Check those symbols for opportunities

## 🎓 Pro Tips

✨ **Combine Filters** - Use multiple filters together for precision  
✨ **Check Liquidity First** - Always verify exchange liquidity before trading  
✨ **Review Correlations** - Avoid portfolio concentration in correlated assets  
✨ **Monitor Volume** - Volume spikes often precede price moves  
✨ **Use Watchlists** - Organize by strategy for faster access
