# UI Prototype — Scanstream

This is a small static HTML/CSS prototype demonstrating the proposed dashboard layout, color palette, and responsive behavior.

Files:
- `index.html` — the static dashboard mock
- `styles.css` — variables and styles

How to open:

1. From PowerShell in the repo root run:

```powershell
Start-Process .\ui-prototype\index.html
```

2. Or open `ui-prototype/index.html` in your browser.

Notes:
- This is intentionally non-interactive and focused on layout and visual hierarchy.
- Next step: an interactive React mock (replay `e2e/smoke-cross-exchange` ticks) or a small Electron wrapper for full-screen monitoring.
